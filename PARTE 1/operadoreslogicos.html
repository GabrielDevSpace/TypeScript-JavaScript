Operadores Lógicos

&& > AND > E

EX: if(num1===1 && num2===2) { resultado = 'num1=1 e num2=2'}

|| > OR > OU

EX: if(num1===1 || num2===1) { resultado = 'num1 ou num2 = 1'}

! > NOT > NAO

EX: if(num1!true) { resultado = 'num1 é falso'}

Economizado codigo com os operadores:

const corUsuario = null
const corPadrao = 'red'
if (corUsuario === null) {
corUsuario = corPadrao
}

Com a correta utilização do operador logico poderiamos simplificar:

const corUsuario = null
const corPadrao = corUSuario || 'red'
