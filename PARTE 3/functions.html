Different Ways to Declare Functions

All declarations will be raised to the top of the documment.

Example:

Function Expression

Can be used as a data or as the value of a variable.

Example:
const iAmAData = Function (){

}

Function useData(value){
value();
}
useData(iAmAData); //I'm using data from another function


Arrow Function

ES6 - ECMAScript6 - New feature

Example:
const iAmAData = () => {
  console.log("Hi")
}
iAmAData(); //less verbose function
